# Testar sem Node.js

## Site online

**URL:** https://onigui.github.io/META-TRAVEL/

### Ativar GitHub Pages (uma vez)

1. https://github.com/Onigui/META-TRAVEL/settings/pages
2. Source: **GitHub Actions**
3. Push na `main` ou rode **Deploy GitHub Pages** manualmente

## Build manual (PC corporativo)

1. https://github.com/Onigui/META-TRAVEL/actions
2. **Build Preview** → Run workflow
3. Baixe artefato **meta-travel-preview**
4. Extraia e abra `pages/index.html`

## Extensão Chrome

Baixe `meta-travel-extension-v1.0.0.zip` em [Releases](https://github.com/Onigui/META-TRAVEL/releases) ou do artefato **chrome-extensions**.
